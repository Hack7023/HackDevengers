# HackDevengers - GenAI Civic Platform

This workspace contains a GenAI-powered civic platform utilizing React (frontend), Node.js (backend gateway), and MongoDB.

## Structure
* `frontend/`: Vite + React + TypeScript + i18next multilingual UI.
* `backend/`: Node.js + Express + MongoDB gateway API.
* `ai-service/`: FastAPI + Python + ChromaDB + Google Gemini RAG pipeline.
* `.agents/`: Agent custom skills and implementation designs.
